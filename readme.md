Tikz-tool
==========

Depends on 
- [tikzjax](http://tikzjax.com) 
- [electron](https://www.electronjs.org)

## Usage
For the first time:
```
npm install
```

Then draw in the `draw.tex` file and run by
```
npm start
```
Maybe you need some time to download resource when you run the tool at the first time.
